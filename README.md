# Agent API SDKs

Public SDKs for the Managed Agent API.

- `javascript/` - JavaScript and TypeScript SDK published as `@agent-api/sdk`
- `python/` - Python SDK published as `cloudsway-agent`
- `go/` - Go SDK module `github.com/scalebox-dev/agent-api-sdk/go`

Each SDK directory contains its own README, changelog, tests, and release notes.
